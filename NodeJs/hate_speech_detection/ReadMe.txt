Abstract:-This is a simple baseline project for hate speech detection using prebuilt sentiment libraries.

pre-requisites:-Set up nodeJS in your system.

Steps to run:-

1.initially download all the modules by calling "npm install" in terminal.

2.run the code by calling "node app.js"

3.search "localhost:8000" in any browser.

4.write any sentence(normal and offending) and press submit.

Result:-It will judge the sentence and give the remark accordingly.
